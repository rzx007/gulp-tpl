基于gulp的前端构建
=================

> [基于gulp的前端开放实践](https://www.gulpjs.com.cn/docs/getting-started/quick-start/)


### 快速开始
#### 安装 gulp 命令行工具
***如果没安装***
```bash
npm install --global gulp-cli
```
`切换到项目目录`
#### 安装依赖
```bash
npm install
```

#### 开发模式

```bash
npm run dev
```
#### 生产模式

```bash
npm run build
```
